Facturación Electrónica con WebPOS

Este módulo permite gestionar la facturación electrónica a través de la interfaz de WebPOS, facilitando la emisión de facturas electrónicas de manera rápida y eficiente.

Tabla de contenidos

  • Uso
  • Seguimiento de Errores
  • Créditos

Uso

  1. Dirígete a Ventas > Punto de Venta y abre una sesión de WebPOS.
  2. Selecciona los productos y procede a la facturación.
  3. Confirma y envía la factura electrónica.

Seguimiento de Errores

Los errores se registran en GitHub Issues. Por favor, revisa si tu problema ya ha sido reportado antes de crear uno nuevo.

Si encuentras un error, ayuda a solucionarlo proporcionando un informe detallado.
